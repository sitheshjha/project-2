# 🍽️ Restaurant Name & Menu Generator

An interactive **web-style Streamlit application** that generates creative restaurant names along with matching menu items based on selected cuisines. The app is designed with a multi-page navigation layout including **Home, Generator, and About sections**, giving it a website-like experience.

This project uses manually defined data for generating names and menus, making it lightweight, beginner-friendly, and easy to understand. It also has future scope for AI/API integration to generate more dynamic and personalized results.

---

## 🚀 Features

- 🏠 Home Page – Overview of the application  
- 🍔 Generator Page – Generate restaurant names & menu items based on cuisine  
- ℹ️ About Page – Project details and information  
- Clean navigation-based web app structure  
- Lightweight and beginner-friendly design  
- Easily extendable for AI/API-based generation in future  

---

## 🛠️ Technologies Used

- Python  
- Streamlit
